# Claude Code Plugins by wyatt727

A collection of plugins for [Claude Code](https://claude.ai/code).

## Installation

Add this marketplace to Claude Code:

```bash
/plugin marketplace add wyatt727/claude-plugins
```

Then browse and install plugins:

```bash
/plugin
# Select "Browse Plugins"
```

Or install directly:

```bash
/plugin install setup@wyatt727
```

## Available Plugins

| Plugin | Description |
|--------|-------------|
| [setup](./setup/) | Project initialization wizard that creates optimized CLAUDE.md files |

## Manual Installation

```bash
git clone https://github.com/wyatt727/claude-plugins.git ~/.claude/plugins/wyatt727
```
